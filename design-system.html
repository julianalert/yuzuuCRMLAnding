Design tokens

The palette is a single neutral “mist” scale defined in OKLCH. Semantic usage on the site pairs mist-950 / white for primary text, mist-700 / mist-400 (dark mode) for secondary copy, and layered translucency for chips, rings, and elevated surfaces.

Layout & spacing

Sections lean on consistent vertical rhythm and a shared content width. Navigation sets scroll offset via CSS variables for anchored sections.

  • max-w-2xl with px-6 by default
  • md:max-w-3xl
  • lg:max-w-7xl lg:px-10
Section shell: py-16 default; alternating bands may use py-24 with bg-mist-50 dark:bg-mist-950/50. Inner stacks often use gap-10 sm:gap-16 between headline blocks and content.
Navbar · sticky header pattern

Sticky bar: sticky top-0 z-10 bg-mist-100 dark:bg-mist-950. Row height aligns with --scroll-padding-top: 5.25rem for scroll-padding-top on html.
